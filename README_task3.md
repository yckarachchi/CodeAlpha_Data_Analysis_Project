A comprehensive data visualization project analyzing IMDb's Top 250 movies. This project transforms raw movie data into insightful visualizations and an interactive dashboard to explore trends, patterns, and relationships in cinematic history.

##  Project Overview

This project demonstrates end-to-end data visualization skills:
- **Data Exploration** of IMDb Top 250 movies dataset
- **Static Visualizations** with Matplotlib and Seaborn (10+ charts)
- **Interactive Dashboard** with Streamlit and Plotly
- **Data Storytelling** and insights generation
- **Portfolio-ready** outputs and documentation
